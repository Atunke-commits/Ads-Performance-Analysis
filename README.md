# Digital Marketing Campaign Analysis
## Project Overview 
This project focuses on analyzing advertising campaign performance data which spans across one month.
The dataset contains **2601** columns and **12** rows, which were initially unclean and inconsistent.

## Objectives
- To evaluate traffic performance
- Financial returns on money spent on Ads
- Conversion Efficiency
- Future marketing predictions
- Insights and Recommendation
## Data Cleaning and Preparation
- Standardized Column formatting
- Removed Inconsistencies and Duplicates
- Accurate Alignment between Clicks, Leads, Conversion, Cost, Sales

## Key Insights
   **Performance Metrics:**
- Average Conversion Rate: **5%**
- Total Leads Generated: **52,000**
- Total Conversions: **$17,00**
**Financial Analysis:**
- Total Sales: **$4M**
- Total Ads Spend: **$559,000**
**Traffic Sources:**
* Mobile devices generated the highest numbers of clicks, accounting for **34%** of total clicks
**Search Insights:**
- The most frequently searched keyword was **ONLINE DATA ANALYTICS**
## RECOMMENDATIONS
- Keyword Targeting
Allocate more budgets towards high performing keywords to maximize engagement
- Optimize For Mobile 
Since Mobile devices drives the most clicks, campaigns should be further optimized for mobile first experiences
- Budget Allocation
Reinvest in channels showing high ROI 
- Implement a Monitoring Alert
Set up an automated system that Immediately flags a sudden drop in sales, to prevent future revenue loss or budget exhaustion
